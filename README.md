# Manikya Alister - Personal Website

Personal academic website built with HTML, CSS, and hosted on GitHub Pages.

## About

This website showcases my research, publications, and professional experience in cognitive science and data science.

## Structure

- `index.html` - Main homepage
- `styles.css` - Stylesheet
- `404.html` - Custom error page
- `sitemap.xml` - SEO sitemap
- `robots.txt` - Search engine crawler instructions
- `static/` - Contains images and PDFs
  - `images/` - Profile photo and hero banners
  - `pdf/` - CV and other documents

## Local Development

Simply open `index.html` in a web browser to view locally.
